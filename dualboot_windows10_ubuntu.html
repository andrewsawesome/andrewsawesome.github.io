How to dual Boot Windows and Ubuntu/Lubuntu

Do I have the right supplies

To dual boot or live boot Linux with Windows, you will need the following things:

  • A Computer with Windows7 or later
  • A drive with 4-8 GB memory(you can tyr one with more memory
  • Internet that your computer can access

What type of OS should I Download

For most recent computers(that aren't running super slow) I would recommend Ubuntu. For a 32 bit computer and/or a computer running Windows Vista or earlier, I would recommend Lubuntu. Next, check your computer. If your computer is 32 bit, you need to download the 32 bit version of the OS. If you're not sure, download the 32 bit version.
