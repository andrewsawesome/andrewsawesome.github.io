How to encrypt Files on Windows 10 Home and Mac

On Windows 10

  1. Download 7-zip by clicking here and clicking Download next to 64-bit x64 from the Download 7-Zip XX.XX (XXXX-XX-XX) for Windows section.
  2. Install 7-zip by double-clicking the .exe file
  3. Open 7-zip File Manager
  4. Navigate to the parent folder of the folder you want to encrypt
  5. Choose the file or folder you want to encrypt.
  6. Click Add
  7. Under Compression Format, choose zip
  8. Under compression Ratio, choose store
  9. Under Encryption Method, choose ZipCrypto
  10. Click "OK"

When the reciever tries to extract or open the file, they will be asked to enter the password.

On MacOS

  1. Download Keka by clicking here, then click the download X.X.X button
  2. Install Keka by double-clicking the file you just downloaded.
  3. Open Keka
  4. Right-click Keka in the dock, then select Options>Keep in dock.
  5. By the close and minimize buttons, click the selection menu and select ZIP.
  6. In the password and repeat sections, enter the password you wish to use for encryption
  7. Open Finder
  8. Select the folder or file you wish to encrypt
  9. Drag this item to the Keka icon in the dock
  10. Select the destination folder
  11. Select "Choose"

    12. When the reciever tries to extract or open the file, they will be asked to enter the password.

    On GNU/Linux(Linux)

    1. Go here, then click DEB x86-64 GTK2(for Debian-based distros) or RPM x86-64 GTK2(for RHEL, Fedora, or CentOS
    2. Install by double-clicking the downloaded .deb or .rpm file.
    3. Open PeaZip and choose the file or folder you would like to encrypt.
    4. Click "Add"
    5. Where it says "Normal", choose "Store"
    6. Click "Enter Password/Keyfile"
    7. Enter the password you would like in the "Password" and "Confirm" sections
    8. Click "OK" on the password dialog box, then click "OK" on the create .zip dialog box.

      9. When the reciever gets the zip file, they will be asked to enter the passsword.
