PDFsam Basic

PDFsam basic is a piece of open source, free software that splits and merges PDFs. Some of the features of this software are:

The ability to rotate the orientation of the pages in a PDF file

The ability to merge different PDF files into 1 cohesive PDF file

Merge two or more PDF files and alternate the pages in reverse or natural order

Extract pages from PDF files

Split a PDF document into multiple files of a given memory size

Download PDFsam Basic here(When installing, be sure to uncheck the checkbox that says "Try PDFSam Enhanced" or something similar)

MIDI Recording

MIDI is a good way to record keyboards and many other musical instruments and even putting multiple parts(like a keyboard track and a drum track) together to create a song.

Aria Maetosa

Aria Maetosa is a free, easy, and good way to record MIDI. Some features of this softawre are:

Put multiple tracks(of the same or different instrument) into one MIDI file

128 keyboard(/guitar?) sounds to pick from

9 types of drum sets to pick from

Download Aria Maetosa here

Video

VLC Media Player

VLC Media Player is a piece of open source, free software that plays and streams audio and video files. Some of the features of this software are:

Play audio and video files from your computer or a flash drive

Play audio and video files from both home-burned and commercial discs(commercial CDs and DVDs)

Stream audio and video files

Get streams of video and audio files(e.g. YouTUbe videos)

Download VLC Media Player Here

URL Shorteners

1. TinyURL

First of all, yes, the TinyURL website has ads. However, using the form below, you can create a TinyURL right from my website. The main standout feature is the ability to create a custom URL without signing up for an account.

Enter a long URL to make tiny:
Custom alias (optional):
https://tinyurl.com/
May contain letters, numbers, and dashes.

2. Rebrandly

Rebrandly's interface is strikingly similar to Bitly's. However, they allow you to track the number of clicks without signing up for an account(to use this, add a + to the end of your shortened link). For a simpler interface, go to rb.gy

3. Bitly

The only reason I would recommend Bitly over Rebrandly is recognition. When you send people a bit.ly link, they're more likely to trust it over an rb.gy link
