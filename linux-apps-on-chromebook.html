How to install Linux apps on Chromebook

This is a tutorial on how linux apps on a chromebook. For this to work, your chromebook must support Linux(Beta).

How to check if your computer is compatible with Linux(Beta) and enable it

  1. Go to the menu on the right and click or tap on the settings cog
  2. Scroll to the bottom of your menu. If you see a Linux(Beta) menu like the next picture, you're good to go.
  3. Click or tap the "turn on" button next to the Linux(Beta) header. You have successfully enabled linux.

The two methods of installing linux apps

1. Installing via terminal

This is the simplest way to install (at least some) Linux apps. This involves the 4 steps below. If you can't install the app via terminal, try one of the other app install methods listed below.

  1. Open the terminal by going into the Chrome OS launcher and finding the terminal
  2. When the terminal opens up, type sudo apt-get update and press enter. If you get an error, don't worry about it and move on to the next step.
  3. After installing any applicable update by pressing y and hitting enter, type sudo apt-get dist-upgrade, and press enter. if you get an error, don't worry about it and move on to the next step. If you get a prompt, installing any applicable update by pressing y and hitting enter.
  4. Now it's time to install the app. Type sudo apt-get install app-name and press enter, replacing app-name with the app you're installing. To install LibreOffice, for example, type sudo apt-get install libreoffice.
