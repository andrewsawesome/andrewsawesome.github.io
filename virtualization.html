Virtaulizing Operating Systems Using Virtualbox/VMWare

Virtualizing operating systems is a great way to try and use operating systems like Windows, Linux, and MS-DOS. Below I have The two most common virtualization softwares and tutorials for installation of a few operating systems inside Virtualboxx/VMware

Virtualbox vs VMWare

The first question you need to answer is which virtualization software you want to use. I recommend Virtualbox because of a few things.

  1. It's free for both consumer and commercial use
  2. It's open source
  3. You can organize your operating systems in groups
  4. It runs on Windows, Linux, and MacOS(The free version of VMWare only runs on Windows and Linux

Click the Virtualbox icon above and choose your operating system to install Virtualbox

The other option is VMWare. You might want to install VMWare for:

  1. Easy Install(VMWare will setup some newer Windows versions automatically)
  2. If you're having problems installing Virtualbox or an operating system in virtualbox, or if a virtual machine will not install in Virtualbox

Install VMWare by clicking the icon above, and choosing your operating system
